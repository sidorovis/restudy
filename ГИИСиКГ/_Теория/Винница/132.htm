 1.3.2. Парабола

 
 

Для параболического интерполятора y=x2 оценочная функция имеет вид uij=xi2-yi. Она разрешает отобразить зависимость, изображенную на рис.1.13. 

Рис. 1.13. Принцип параболической интерполяции по методу оценочной функции

При U³0 выполняют шаг по оси X, а при U<0 – пo ocи Y. При выполнении шага по оси Х новое значение оценочной функции рассчитывают согласно выражения

y = x2.

При выполнении шага вдоль оси Y значение оценочной функции определяют как

uij = xi2 – yi.

Таким образом, параболический интерполятор – это круговой интерполятор с одним заблокированным блоком для расчета оценочной функции.
